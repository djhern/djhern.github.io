GradChain is a web application that I have been developing with a team as a school project.
Utilizing blockchain technology to cryptographically verify documents, GradChain allows universities to upload student diplomas for convenient verification. Employers searching for candidates can then view a student’s diploma or credentials through the website, knowing each document is authentic and the candidate is qualified.

How does it work?

The Interplanetary File System (IPFS) is essentially a blockchain database that stores files as fragments of data on thousands of devices and retrieves the file by rebuilding each fragment through a Content Identifier (CID). By uploading a file to IPFS, GradChain can verify the untampered and genuine nature of the file for the employer. Click below to view the application’s GitHub repository.
